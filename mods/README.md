# Папка для модов

Поместите сюда .jar файлы ваших модов для Minecraft 1.12.2.

## Важно:
- Все моды должны быть совместимы с Minecraft 1.12.2
- Моды должны быть совместимы с Forge 14.23.5.2860 или близкими версиями
- Убедитесь, что моды совместимы между собой

## Рекомендуемые моды:
- JEI (Just Enough Items)
- Biomes O' Plenty
- Tinkers' Construct
- Applied Energistics 2
- И другие популярные моды для 1.12.2

После добавления модов закоммитьте изменения, и Railway автоматически пересоберет сервер.
